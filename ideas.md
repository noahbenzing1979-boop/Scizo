# Prismatic Depths — Design Ideas

## Idea 1: Arcane Manuscript
**Design Movement:** Dark Fantasy Illuminated Manuscript
**Core Principles:** Parchment textures, hand-drawn dungeon aesthetics, runic typography, candlelit warmth
**Color Philosophy:** Deep obsidian backgrounds with amber/gold accents, prismatic jewel tones for dice (ruby, sapphire, emerald), blood-red danger indicators
**Layout Paradigm:** Asymmetric split — left panel shows dungeon map/log, right panel shows hero stats and dice rolling area
**Signature Elements:** Glowing rune borders, animated dice with 3D perspective, dungeon-tile grid map
**Interaction Philosophy:** Every button press feels like casting a spell — glowing hover states, particle effects on dice rolls
**Animation:** Dice tumble with physics-like animation, damage numbers float upward, rooms "reveal" with a fog-of-war dissolve
**Typography:** Cinzel Decorative for headers, Crimson Text for body — medieval but readable

## Idea 2: Neon Dungeon Terminal
**Design Movement:** Cyberpunk meets Dungeon Crawler — Dark terminal aesthetic with neon prismatic accents
**Core Principles:** Dark slate backgrounds, neon glow effects, monospace data readouts, prismatic color spectrum for dice
**Color Philosophy:** Near-black (#0a0a0f) base with prismatic neon accents cycling through the spectrum — each die type has its own color (D4=violet, D6=blue, D8=cyan, D10=green, D12=amber, D20=crimson)
**Layout Paradigm:** Vertical split — top half is dungeon action log (scrolling terminal), bottom half is the interactive dice/action panel. Hero cards float as side panels
**Signature Elements:** Scanline overlay texture, glowing neon dice, ASCII-art dungeon map
**Interaction Philosophy:** Actions feel like executing commands — typewriter text effects, glitch animations on PRISM failures
**Animation:** Dice roll with neon trail effects, PRISM failure triggers a screen glitch/shake, XP banking pulses with a prismatic wave
**Typography:** Space Grotesk for UI, JetBrains Mono for numbers/dice — clean but technical

## Idea 3: Stained Glass Cathedral
**Design Movement:** Gothic Stained Glass — Each game element is a jewel-toned panel
**Core Principles:** Rich jewel colors, lead-line borders between panels, light-through-glass luminosity, cathedral grandeur
**Color Philosophy:** Deep midnight blue base, with each depth having its own jewel palette — Depth 1 (amber/gold), Depth 2 (sapphire/teal), Depth 3 (ruby/crimson). Dice glow like colored glass
**Layout Paradigm:** Mosaic panel layout — the game screen is divided into stained-glass-style panels of varying sizes, each containing a game element
**Signature Elements:** Lead-line borders (thin dark dividers between colored sections), radial glow effects, prismatic light rays
**Interaction Philosophy:** Interactions feel sacred and weighty — slow, deliberate animations, choral-like transitions
**Animation:** Panels illuminate when active, dice cast colored light on the table, depth transitions use a stained-glass shatter/reassemble effect
**Typography:** Playfair Display for headers, Lora for body — elegant and timeless

## Selected Design: Neon Dungeon Terminal

The Neon Dungeon Terminal aesthetic was chosen for its perfect alignment with the PRISM dice system — each die type glows in its own spectral color, making the press-your-luck mechanic visually intuitive. The dark terminal background keeps focus on the dice and action log, while neon accents create the "prismatic" feel referenced in the game's name. The typewriter/glitch effects make PRISM failures feel dramatic and memorable.
